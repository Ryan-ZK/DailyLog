# 工作打卡 App

一个自用的移动端优先打卡工具，用来记录固定类型工作的开始和结束时间，并在日历和统计面板里查看结果。

## 使用方式

直接用浏览器打开 `index.html`。

数据会保存在当前浏览器的本地存储里，不会上传到服务器。

作为 iPhone 主屏幕 App 使用时，需要通过本地服务器或线上地址打开页面，然后在 Safari 中选择“分享” -> “添加到主屏幕”。直接用 `file://` 打开时，离线缓存不会注册，这是浏览器限制。

## 当前功能

- 底部导航拆分为首页、日历、统计三页
- 支持 PWA 配置，可添加到 iPhone 主屏幕
- 首页展示当日日期
- 开始/结束计时
- 自动记录开始时间、结束时间和时长，精确到秒
- 日历连续平铺展示，并可一键回到本月
- 统计近 7 天、近 30 天、近 90 天、本月、全部记录
- 查看最近 20 条记录
- 删除单条记录
- 导出和导入 JSON 备份
